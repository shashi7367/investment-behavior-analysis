# Investment Behavior Analysis Dashboard

This project was completed as part of my internship at **Cognifyz Technologies**. The goal of this project was to analyze investment behavior data using **Power BI** and generate meaningful insights through interactive dashboards.

## Project Overview

The dataset contains information related to:

* Age of investors
* Gender
* Investment preferences
* Savings objectives
* Investment duration
* Monitoring frequency
* Reasons for investment
* Sources of investment information

The project focuses on understanding how individuals make investment decisions and identifying trends through data visualization.

---

## Tasks Completed

### Task 1: Data Exploration and Summary

* Imported dataset into Power BI
* Explored data types and structure
* Calculated:

  * Average age
  * Investment percentage
  * Common savings objectives
* Created summary visualizations

---

### Task 2: Gender-Based Analysis

* Compared male vs female investment preferences
* Analyzed differences in equity, mutual funds, and other investments
* Generated gender comparison charts

---

### Task 3: Objective Analysis

* Studied relationship between savings objectives and investment choices
* Compared objectives such as:

  * Capital Appreciation
  * Growth
  * Income
* Built charts for objective-based investment analysis

---

### Task 4: Investment Duration and Frequency Analysis

* Analyzed investment duration patterns
* Studied monitoring frequency:

  * Daily
  * Weekly
  * Monthly
* Visualized investment behavior trends

---

### Task 5: Reasons for Investment

Analyzed reasons behind investment choices such as:

* Better Returns
* Tax Benefits
* Capital Appreciation
* Safety
* Fixed Returns

Created visual charts for each investment avenue.

---

### Task 6: Source of Information Analysis

Analyzed where people gather investment information from:

* Financial Consultants
* Newspapers
* Television
* Internet

Identified the most trusted sources of investment information.

---

### Task 7: Final Dashboard

Built a comprehensive interactive dashboard combining all insights:

* KPI Cards
* Bar Charts
* Donut Charts
* Slicers
* Interactive Filters

---

## Tools Used

* Power BI

## Key Insights

* Most investors prefer mutual funds
* Retirement is the most common savings objective
* Monthly investment monitoring is most common
* Financial consultants are the most trusted source of information
* Capital appreciation is a major reason for equity investment
